About

Explore the Mandelbrot Set.

Using this web application you can explore the Mandelbrot Set. My first images of the Mandelbrot Set I computed on a Sinclair ZX Spektrum using HiSoft Pascal 4.
This is my second try - but mainly this is about toying around with some features of Spring Boot, Spring Framework, neo4j, Bootstrap and Thymeleaf.
